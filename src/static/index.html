CalVAD: California Vehicle Activity Database

This is the primary interface to the California Vehicle Activity Database. The goal is to use observed data to estimate the actual hourly vehicle movement in the state. While there are limitations to depending solely on data, this offers a unique counter-point to model-based approaches.

Map Viewer

The map interface provides a way to view the daily estimates of vehicle activity for each 4km grid cell covering California. The map demonstrates the breadth and scope of the kinds of data CalVAD has generated from the source inputs.
